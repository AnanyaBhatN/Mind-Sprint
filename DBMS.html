Database Management System

A Database Management System (DBMS) is a software application that interacts with end-user, other applications, and the database itself to capture and analyze data.

STUDY MATERIALS

  • SQL

    A language designed for managing and manipulating data in relational databases. It's used to interact with databases, allowing users to retrieve, insert, update, and delete data.

    PDF

  • PL/SQL

    A procedural extension of SQL. It combines the power of SQL with programming constructs, allowing developers to write complex database applications and automate tasks.

    PDF

  • RELATIONAL DATABASE

    Organizes data into tables with rows and columns. These tables are linked together through relationships, allowing for efficient data storage and retrieval. This structure makes it easy to manage and query complex information.

    PDF

  • NO SQL

    NoSQL databases offer a flexible approach to data storage and retrieval, particularly for large and unstructured data sets.

    PDF

  • ER MODEL

    A high-level data model used to design databases. It visually represents entities (objects or concepts) and relationships between them.
